GStreamer Utilities (savant_rs.gstreamer)
=========================================

The GStreamer utilities are part of the ``savant_rs`` package and are
available when ``savant_rs`` is built with the ``gst`` Cargo feature.

.. note::

   These bindings were previously shipped as the standalone
   ``savant_gstreamer_py`` crate.  They have been consolidated into
   ``savant_rs.gstreamer`` as of version 1.15.
