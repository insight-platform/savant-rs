DeepStream NvBufSurface (savant_rs.deepstream)
==============================================

The DeepStream NvBufSurface bindings are part of the ``savant_rs`` package
and are available when ``savant_rs`` is built with the ``deepstream`` Cargo
feature (which automatically enables the ``gst`` feature).

.. note::

   These bindings were previously shipped as the standalone
   ``deepstream_nvbufsurface`` Python package.  They have been consolidated
   into ``savant_rs.deepstream`` as of version 1.15.
